# Backend (Express)

Basic Express backend for the AuthProject.

Setup

```bash
cd Backend
npm install
```

Run

```bash
npm run dev   # requires nodemon
# or
npm start
```

Environment

- Copy `.env.example` to `.env` and adjust `PORT` as needed.

API

- `POST /api/auth/signup` - signup stub
- `POST /api/auth/login` - login stub
